An end-to-end supply chain analytics project built using Python and SQL — covering KPI analysis, supplier scoring, inventory risk, and demand forecasting across 2,000 orders and 5 suppliers over 16 months.

---

## 🛠️ Tech Stack
Python · Pandas · NumPy · SQLite3 · Matplotlib · Seaborn

---

## 📊 Key Results

| KPI | Value |
|---|---|
| Total Revenue | ₹52 Crore |
| On-Time Delivery | ~50% |
| Fulfillment Rate | 80% |
| Forecast Accuracy | ~91% |
| High Stockout Products | 4 |

---

## ⚠️ Critical Findings
- **SKU-007** has only **2 days of stock** remaining — immediate reorder needed
- **Supplier B** flagged At Risk — lowest OTD and highest defect rate
- **4 products** below reorder point across Electronics and Apparel
